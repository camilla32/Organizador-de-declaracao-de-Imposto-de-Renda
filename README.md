Organizador de Declaração de Imposto de Renda
--------

O projeto consiste na criação de um ferramenta no Excel para ajudar a organizar e reunir informações para a declaração do Imposto de Renda a partir de aulas da Dio com Professor Felipe no 
Bootcamp de Excel da Santander. O nome dado a ferramenta foi App controle Lion.
